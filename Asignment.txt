 10. How can we club two or more rows or columns into a single row or column in an
HTML table? 

We use the colspan and rowspan attribute, to merge cells in HTML.

 The rowspan attribute is for the number of rows a cell should merge, whereas the colspan attribute is for the number of columns a cell should merge. 

Example:<!DOCTYPE html>
<html lang="en">

<head>
   <meta charset="UTF-8">
   <meta http-equiv="X-UA-Compatible" content="IE=edge">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Document</title>
</head>

<body>
   <table border="1" cellspacing="0.5" cellpadding="5">
      <tr>
         <!-- <td colspan="4" align="center">1</td> -->
         <th colspan="4"> 1 </th>
         <!-- <td>2</td>
         <td>3</td>
         <td>4</td> -->
      </tr>
      <tr>
         <td rowspan="3">1</td>

         <td>3</td>

      </tr>

         <td>1</td>

      </tr>

Two commonaly used block elements are: <p> <h1> <div>

Inline-Element

 An inline element does not start with new line and take width as per requirement.
